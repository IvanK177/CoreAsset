# Руководство пользователя CoreAsset

---

## Вход в систему

1. Перейдите по адресу приложения
2. Введите **email** и **пароль**
3. Нажмите **Войти**

> Учётные записи создаются администратором. Самостоятельная регистрация недоступна.

## Дашборд

Главный экран показывает:
- Общее количество рабочих мест / вакантных / в ремонте
- Процент исправного оборудования
- Критические тикеты (требуют внимания)
- Лицензии, истекающие в течение 30 дней

## Работа с компьютерами

### Добавить компьютер

1. Раздел **Компьютеры** → кнопка **+ Добавить**
2. Заполните обязательные поля: Инвентарный номер, Тип, Кабинет, Статус
3. При необходимости — заполните характеристики (CPU, RAM и т.д.)
4. Нажмите **Сохранить**

### Изменить статус

Откройте карточку компьютера → поле **Статус жизненного цикла** → выберите новое значение → **Сохранить**.

Доступные статусы: **Активен · В ремонте · Вакантен · Списан**

### Привязать сотрудника

Карточка компьютера → раздел **Рабочее место** → выберите сотрудника из списка → **Назначить**.

### Удалить компьютер

Карточка компьютера → кнопка **Удалить** → подтвердите в появившемся окне.

> ⚠️ Удаление компьютера также удаляет все связанные тикеты (каскад). Установленное ПО открепляется.

---

## Работа с сотрудниками

- **Добавить:** раздел **Сотрудники** → **+ Добавить** → заполните ФИО, должность, email
- **Уволить:** карточка сотрудника → переключить статус на **Уволен** → рабочее место автоматически становится «Вакантным»

---

## Управление лицензиями

### Добавить пул лицензий

Раздел **Лицензии** → **+ Добавить** → выберите ПО из справочника → укажите тип (подписка / бессрочная), лимит мест, дату окончания.

### Привязать ПО к компьютеру

Карточка компьютера → раздел **Установленное ПО** → **+ Добавить ПО** → выберите пул.

> Если пул исчерпан (использовано = лимит), добавление заблокировано.

### Алерты истечения

Пулы с подпиской, истекающей **менее чем через 30 дней**, выделяются красным в таблице лицензий.

---

## Журнал инцидентов

### Зарегистрировать инцидент

Карточка компьютера → раздел **Инциденты** → **+ Создать тикет** → заполните тип, описание, приоритет.

### Жизненный цикл тикета

```
Открыт  →  В работе  →  Исправлен
```

Изменить статус: откройте тикет → кнопка **Изменить статус**.

> Закрытые (Исправлен) тикеты **не могут быть удалены** — они остаются в архиве навсегда.

## Поиск и фильтрация

- Строка поиска (верхняя панель): поиск по **инвентарному номеру, ФИО, IP, MAC-адресу**
- Фильтры в таблице компьютеров: **Статус**, **Кабинет**
- Сортировка: нажмите на заголовок любого столбца таблицы
