## Key Business Algorithms

### Algorithm 1 вҖ” License Seat Assignment

```
User clicks "+ Add Software" on computer card
    в”Ӯ
    в–ј
Fetch license_pools WHERE software_id = selected
    в”Ӯ
    в”ңв”Җв”Җ used_seats >= total_seats?
    в”Ӯ       YES вҶ’ Show error: "License pool exhausted"
    в”Ӯ       NO  вҶ“
    в–ј
INSERT into software_installations (computer_id, license_pool_id)
    в”Ӯ
    в–ј
UPDATE license_pools SET used_seats = used_seats + 1
    в”Ӯ
    в–ј
Revalidate page вҶ’ show updated pool balance
```

### Algorithm 2 вҖ” Employee Dismissal (cascade to workplace)

```
Admin sets employee status вҶ’ "Fired"
    в”Ӯ
    в–ј
UPDATE employees SET is_active = false WHERE id = ?
    в”Ӯ
    в–ј
DB constraint: ON DELETE SET NULL on workplaces.employee_id
вҶ’ workplace.employee_id = NULL automatically
вҶ’ computer lifecycle_status вҶ’ "Vacant"
    в”Ӯ
    в–ј
Revalidate /employees and /computers pages
```

### Algorithm 3 вҖ” License Expiry Alert

```
On page load of /licenses:
    в”Ӯ
    в–ј
SELECT * FROM license_pools
WHERE license_type = 'subscription'
AND expires_at <= NOW() + INTERVAL '30 days'
    в”Ӯ
    в–ј
For each result вҶ’ render red badge in table row
```
